# xbox-query

用于查询xbox玩家信息的 NodeJS 包

## 使用

1. 通过CLI

    1. 安装npm包

        ```bash
        npm install -g xbox-query
        ```

    2. 登录你的xbox账号

        ```bash
        xbox-query login
        ```

    3. 查询玩家

        ```bash
        xbox-query query < gamer_tag | xuid >
        ```
